초기 기획 :
 클래스 레스토랑 : 
- input : 위치, 식당이름, 메뉴

기능 :
  1. 날짜 입력
  2. 메뉴추가 삭제
  3 . 위치와 식당이름, 메뉴들을 인풋으로 받아 프린트. __str___
 
   4.  해당 위치와 식당이름을 기반으로 네이버 검색해서 리뷰를 통해 식당 분위기, 메뉴 등을 크롤링
	4-1 식당에 대한 전반적인 리뷰 크롤링 (텍스트 뭉치)
        4-2 메뉴에 대한 리뷰 크롤링
   5. 2가지 크롤링한 정보를 요약하는 gemini api 프롬프트 생성 및 리뷰 요약 결과

클래스 블로그 글 생성 :
     1. 위에서 생성한 기본정보 및 리뷰요약 정보를 가져옴
     2. 정보를 종합한 제목 생성
     3. 블로그 개요 생성
     4. 본문 생성
     5. 결론 생성

클래스 포스팅
  : 위에서 생성한 결과를 실제로 블로그에 포스팅
